## Copilot session history: restore archived chats and resume agent tasks

### 1) GitHub Copilot Chat on GitHub.com (primary)
1. Open this repository on GitHub.com.
2. Click the **Copilot** button to open the chat panel.
3. Open **Conversation history** (clock/history icon in the chat panel).
4. Go to **Archived** conversations.
5. Select the conversation you want, then choose **Unarchive/Restore**.
6. Reopen that conversation and continue chatting in the same thread.

### 2) Copilot Chat in VS Code (quick reference)
1. Open the **Copilot Chat** view in VS Code.
2. Open chat **History** from the chat header/menu.
3. Find archived/older conversations and reopen one.
4. Continue in the restored thread.

> UI labels can vary slightly by Copilot version, but the flow is: **Chat → History → Archived → Restore → Continue**.

### 3) Copilot coding agent task/PR sessions (GitHub)
To continue work from an earlier Copilot coding agent task:

1. Open the repo **Pull requests** tab.
2. Locate the prior Copilot task PR (common pattern: `copilot/...` branch or Copilot-generated title).
3. Open that PR and review the existing discussion/checks.
4. Continue in the same PR thread (add your follow-up request in the PR conversation).
5. If you need the original task entry point, open **Actions** → the related **Running Copilot cloud agent** run → follow the linked PR.

This keeps context in the same task thread instead of starting a brand-new PR/session.
