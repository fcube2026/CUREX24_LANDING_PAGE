# CUREX24 Landing Page – Help & FAQ

## Archived agents / conversations: how to reopen and continue

In this project, **“archived”** means a chat session that is hidden from the active list but not deleted.

### In the project UI (floating chat button)

1. Open the chat popup from the **💬 button** (bottom-right).
2. Select the **Archived** tab.
3. Find the chat you want and click **Restore**.
4. Go back to **Active** and click **Continue** to resume it.

### In GitHub Copilot Chat on GitHub.com

If by “archived agent” you mean a previous Copilot chat/conversation:

1. Open **GitHub Copilot Chat** in GitHub.
2. Open your **conversation history** (sidebar/history list).
3. Locate archived/past conversations.
4. Open the conversation and use the available action to **reopen/unarchive** (wording can vary by UI version).
5. Continue chatting in that restored thread.

### Notes and limitations

- UI labels can differ slightly by GitHub Copilot version and account features.
- Some older conversations may be read-only or unavailable if retention limits apply.
- “Delete” is permanent; archived conversations can usually be restored, deleted ones cannot.
